How to Add Multiple Images to Products

1. Structure for Multiple Images

Each product can now have an images array with the following structure:

{
  "images": [
    {
      "id": "img1",
      "details_desktop": "url-for-large-desktop-view.jpg",
      "details_mobile": "url-for-mobile-view.jpg", 
      "products_desktop": "url-for-products-page-desktop.jpg",
      "products_mobile": "url-for-products-page-mobile.jpg",
      "alt_text": "Description of the image",
      "order": 1
    },
    {
      "id": "img2",
      "details_desktop": "url-for-second-image.jpg",
      "alt_text": "Description of second image", 
      "order": 2
    }
  ]
}

2. Image Size Recommendations

  • details_desktop: 800x800px or larger (for product details page)
  • details_mobile: 600x600px (optimized for mobile details page)
  • products_desktop: 400x400px (for products listing page)
  • products_mobile: 300x300px (for mobile products listing)

3. Features

  • ✅ Products Page: Shows only the first image (ordered by 'order' field)
  • ✅ Product Details Page: Shows all images with navigation
  • ✅ Manual Navigation: Users can click thumbnails or use arrow buttons
  • ✅ Keyboard Navigation: Left/Right arrow keys
  • ✅ Responsive: Different images for desktop and mobile
  • ✅ Fallback: Works with existing single-image products
